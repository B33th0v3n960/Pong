# Pong Game

build with [Processing](processing.org)

## Collision Detection System
- Collision detection system with Separation Axis Theorem (SAT)
- Detects collision between: 
    - circle to circle
    - rectangle to rectangle (including rotation)
    - rectangle to circle (including rotation)

## The Game
Using the collision system, created a simple pong game to test out the collision
system. 
- Player vs Computer setup
- Acceleration based on location of the bounce
- Angling base on the location of the bounce
